3.52c - экспериментальная версия, запуск под Windows рекомендуется делать опытным 
	пользователям. 


	Что нового:

+ Исправлен глюк с возвратом неверного значения HPA на некоторых винтах, это 
  моя ошибка. Все еще раз перепроверено.
+ Добавлены команды HPA48, NHPA48 для принудительного указания нужного режима;
+ Протокол "проверки интерфейса" подправлен в соответствии с АТА-стандартом, 
  для минимизации ошибок на SATA-II.
+ Споссоб подачи команд изменен для совместимости с дополнительными ATA 
  контроллерами в Windows XP. Скорость работы должна увеличиться;
+ Произведена отладка программы под драйвером PortTalk 
  (http://www.beyondlogic.org/porttalk/porttalk.htm) в Windows XP SP1,
  для работы с дополнительными ATA/SATA контроллерами непосредственно из XP.
  Образец ярлыка для запуска Victoria с PortTalk прилагается;
+ Добавлена поддержка еще 17 дополнительных SATA контроллеров Promise и 
  nVidia (Информация предоставлена: Роман aka bootrom);
+ Добавлена фирма-изготовитель ALi;
+ Добавлен новый смарт-атрибут 190;
- Убрана поддержка temporary HPA;
